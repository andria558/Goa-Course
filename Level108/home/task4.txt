JSX (JavaScript XML) არის სინტაქსური გაფართოება JavaScript-ისთვის, რომელიც გამოიყენება React-ში UI კომპონენტების შესაქმნელად.

JSX-ის მეშვეობით შესაძლებელია HTML-ის მსგავსი სტრუქტურის დაწერა პირდაპირ JavaScript კოდში. მაგალითად:

const element = <h1>Hello, world!</h1>;

ეს კოდი რეალურად გადაიქცევა JavaScript ფუნქციად:
React.createElement('h1', null, 'Hello, world!');

JSX-ის მახასიათებლები:
1. HTML-ის მსგავსი სინტაქსი
2. გამოიყენება React კომპონენტებში
3. უნდა ჰქონდეს ერთი გარე ელემენტი
4. ატრიბუტები იწერება HTML-ის მსგავსად
5. JSX კოდი ტრანსპილირდება Babel-ის საშუალებით

JSX ამარტივებს UI-ის აღწერას და კოდის წაკითხვას React პროექტებში.
