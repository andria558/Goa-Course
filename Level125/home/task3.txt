TailwindCSS რესპონსივი Breakpoints (Georgian)

TailwindCSS მუშაობს Mobile-First პრინციპით, რაც ნიშნავს რომ სტილები პირველ რიგში გამოიყენება ყველაზე პატარა ეკრანებზე და შემდეგ იზრდება ზომის მიხედვით.

1. Default (მობილური)
   - როდესაც კლასს არ აქვს მითითებული breakpoint (მაგ. text-sm), ის გამოიყენება ყველა ეკრანზე, განსაკუთრებით მობილურზე.
   - ეს არის საწყისი სტილი.

2. sm (პატარა ეკრანები) ≥ 640px
   - გამოიყენება პატარა ტაბლეტებზე ან მცირე ზომის მოწყობილობებზე.
   - მაგ: sm:text-lg → ტექსტი გახდება დიდი 640px-ზე მეტი ეკრანისთვის.

3. md (საშუალო ეკრანები) ≥ 768px
   - გამოიყენება ტაბლეტებზე და პატარა ლეპტოპებზე.
   - მაგ: md:grid-cols-2 → 2 სვეტი 768px-ზე მეტი ეკრანისთვის.

4. lg (დიდი ეკრანები) ≥ 1024px
   - გამოიყენება ლეპტოპებზე და სტანდარტულ მონიტორებზე.
   - მაგ: lg:flex → Flex layout 1024px-ზე მეტი ეკრანისთვის.

5. xl (ძალიან დიდი ეკრანები) ≥ 1280px
   - გამოიყენება დიდ მონიტორებზე.
   - მაგ: xl:text-2xl → ტექსტი გახდება უფრო დიდი 1280px-ზე მეტი ეკრანისთვის.

6. 2xl (სუპერ დიდი ეკრანები) ≥ 1536px
   - გამოიყენება ძალიან ფართო ეკრანებზე ან ტელევიზორებზე.
   - მაგ: 2xl:container → კონტეინერი მიიღებს მაქსიმალურ სიგანეს 2xl ეკრანებზე.

შენიშვნა:
Tailwind-ის ყველა responsive კლასი იწყება breakpoint-ით და შემდეგ მოყვება სტილი.
მაგალითად: md:bg-red-500 ნიშნავს რომ ფონი გახდება წითელი მხოლოდ 768px-ზე მეტი ეკრანისთვის.
