useEffect-ში ორი მთავარი კონცეფციაა:

1. Cleanup function
   - ეს არის ფუნქცია, რომელსაც useEffect აბრუნებს.
   - გამოიყენება რესურსების გასუფთავებისთვის, როცა კომპონენტი იხურება ან effect ხელახლა გაეშვება.
   - მაგალითად: event listener-ის მოხსნა, interval-ის გაჩერება, subscription-ის გაუქმება.
   - მიზანი: თავიდან ავიცილოთ memory leak და ზედმეტი პროცესები.

2. Dependency array
   - ეს არის მასივი, რომელიც განსაზღვრავს როდის გაეშვება useEffect.
   - [] (ცარიელი) ნიშნავს, რომ effect მხოლოდ ერთხელ გაეშვება (componentDidMount-ის მსგავსად).
   - თუ მასივში ჩავწერთ state ან prop-ს, effect იმუშავებს მხოლოდ მაშინ, როცა ეს მნიშვნელობა შეიცვლება.
   - თუ საერთოდ არ დავწერთ dependency array-ს, effect გაეშვება ყოველ render-ზე.
